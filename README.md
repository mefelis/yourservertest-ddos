## ✨ Features

- 🎯 **Two attack modes**: Burst (all at once) or Sustained (RPS limit)
- 📊 **Real-time progress tracking** with visual indicators
- 🔄 **Auto-restart** for multiple test rounds
- ⚡ **Lightweight** - built with Node.js
- 🛡️ **Safe** - only works on servers you specify
